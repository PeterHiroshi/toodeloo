# Toodeloo

This is some playing around with Box2D with the goal of creating a platform game.

## Requirements:

  * cmake
  * OpenGL
  * [Box2D](http://www.box2d.org/)
  * SDL

## To compile and run:

Follow the instructions in src/box2d/INSTRUCTIONS, and then do:

    cd src
    cmake .
    make
    ./game
